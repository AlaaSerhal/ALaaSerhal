Latest Projects

SOS: Smart Obviation System

A Smart Fire Prevention System

New

  Fire is one of the most dangerous risks that continuously threatens our safety; the damages accompanied by these risks led to high demands on the invention and continuous development of firefighting systems. In light of this observation, this project provides firefighters and private facilities with a powerful tool that can detect, prevent, and extinguish fires in the context of in/out-door environments, while preserving un-burned areas and appliances. More specifically, this objective is achieved through the means of computer vision and image processing operations, which are supplemented with an in-house, directive firefighting sprinkler. SOS provides a revolutionized and intelligent solution for the current fire systems by eliminating the possibility of having false alarms while minimizing water damages, maintenance costs and resource consumption.

State - Space Search Vacuum Cleaner Agent

Our agent would govern the behavior of a vacuum cleaner operating in an environment consisting of a tiled surface made of 'NxM' tiles. The agent would accept this grid as an input with the presence of some dirty tiles that may also have boundaries that can’t be traversed. The agent would be able to clean, automatically and autonomously, the tiles in the optimal and maximum performance. Multiple cleaning and dirt agents can be deployed in different PEAS. Several algorithms were used, most importantly genetic alogrithm, Dijkstra (shortest path) and inverse PAC-man.
